## About The Project

SystemFlow is a automation/organization tool for your files. SystemFlow can help you organize, locate, and clean up your file system. SystemFlow aims to become an automation engine for your file system Currently, SystemFlow offers the following:
- Find any file via the search engine
- Full System Scanning - index files across entire drive
  

Later version will include new features
-Smart Cleanup tools
-Rule-based automations
-Drive usage visualization
